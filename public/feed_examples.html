output options

There a few way to access your lists. The urls spesific to your lists can be found on your lists page after you login.

RSS Feed:

Each list has a RSS Feed assoicated with it. The RSS feed contains the 25 most recent notes that arn't marked as completed.
Firefox has great native handling for using RSS in this manner or if you use chrome I would suggest Foxish(full disclosure: I wrote it). Or finally Google Reader.

Readable Url:

Each list has an optional readable url. You can disable this by unchecking the readable url option for the list.
This url takes the form: http://notefeeder.heroku.com/<your user name>/<your lists name>

Plain Url:

This is slightly more private and its just an auto generated id for the list. This url hides who owns the list.
