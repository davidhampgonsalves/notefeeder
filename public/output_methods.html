output options

There a few way to access your lists. Each of these urls can be found on your lists page beside each list.

RSS Feed:

Each lists as an RSS Feed assoicated with it. The RSS feed contains the newest 25 notes that arn't marked as completed.
Firefox has great native handling for this or if you use chrome I would suggest Foxish(full disclosure: I wrote it). Or finally Google Reader.

Readable Url:

Each list has an optional readable url. You can disable this by unchecking the readable url option for the list.
This url takes the form: http://notefeeder.heroku.com/<your user name>/<your lists name>

Private Url:

This is slightly more private and its just an auto generated id for the list. This url hides who owns the list.
This url takes the form: http://notefeeder.heroku.com/lists/<list id>
