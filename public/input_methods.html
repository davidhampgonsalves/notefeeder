input methods

bookmarklet

The main and easiest way to create notes in notefeeder is using the bookmarklet. After you login your bookmarklet will be at the bottom of the page. To save it for later just drag the link to your bookmark bar.

Once it is safely in your bookmark bar you can click on it without leaving your current page you'll see an area slide in which will already have the current page url and title populated. Then you can make any changes or add some details and create the note.

get request

You can also create note using a simple get request to the url http://notefeeder.heroku.com/create/notes. This request must contain the list_id whichis is the number you see in the lists private url. It must also contain at least one of the following: title, url or description.
ie: http://notefeeder.heroku.com/create/notes?list_id=1&url=notefeeder.heroku.com

This can be useful if you use firefox and create a bookmark keyword on this url. This will let you add simple notes from your address bar.
ie: http://notefeeder.heroku.com/create/notes?list_id=<your list id>&url=%s

plain old form

You of course can also just login to notefeeder and click "add note" on any of your lists to create whatever you need.
